-------------------------Paso 1: Abriendo el Backend-------------------------

1.Abre una terminal en tu computadora.

2.Ingresa a la URL del backend utilizando el comando cd finalIME\api-cobot. Esto te llevará al directorio donde se encuentra el código fuente del backend.

3.Una vez dentro del directorio del backend, ejecuta el servidor utilizando el comando python manage.py runserver. Esto iniciará el servidor del backend y estará listo para atender solicitudes.

-------------------------Paso 2: Abriendo la Interfaz de Usuario-------------------------

1.Abre otra terminal en tu computadora.

2.Ingresa a la URL de la interfaz de usuario utilizando el comando cd finalIME/Front-ARIA. Esto te llevará al directorio donde se encuentra el código fuente de la interfaz de usuario.

3.Antes de ejecutar la aplicación, asegúrate de instalar las dependencias necesarias utilizando el comando npm install.

4.Una vez que se completen las instalaciones, ejecuta la aplicación utilizando el comando npm start. Esto iniciará la interfaz de usuario y abrirá una ventana del navegador web donde podrás interactuar con la aplicación.

Siguiendo estos pasos, habrás abierto y levantado tanto el backend como la interfaz de usuario de tu proyecto. El backend estará escuchando solicitudes en la URL proporcionada, y la interfaz de usuario se abrirá en un navegador web para que puedas interactuar con la aplicación. ¡Listo para empezar!
